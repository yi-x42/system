React 狀態檢查

請在瀏覽器控制台中檢查以下項目:

  1. 打開開發者工具 (F12)
  2. 切換到 Console 頁面
  3. 查看是否有錯誤訊息
  4. 執行以下指令檢查狀態:
// 檢查模型狀態
window.reactDevTools = true;
console.log('檢查 React Query 快取...');

// 如果使用 React Query DevTools，可以執行:
// window.__REACT_QUERY_STATE__ 

console.log('檢查 selectedModel 狀態...');
// 在 React 組件中添加 console.log 來查看狀態
